- `1` - player1,
- `2` - player2,
- `0` - empty field

|-------|-------------|-------------|-------------|

| row 1 | `board[1][0]` | `board[1][1]` | `board[1][2]` |
| row 3 | `board[2][0]` | `board[2][1]` | `board[2][2]` |
